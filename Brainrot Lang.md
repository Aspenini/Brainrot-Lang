## Braincells
Data can be stored in any of the 7 braincells:  

- `rizz`  
- `aura`  
- `peak`  
- `goon`  
- `mog`  
- `npc`  
- `sigma`  

Example:  

```brainrot
FANUMTAX peak FR 2
```

---

## I/O
`SAY` will print values.  

```brainrot
FANUMTAX goon FR 69
SAY goon
/// 69
```

---

## Data and Copy
- `FANUMTAX` declares/assigns a value to a braincell.  
- `DIDDLE` copies the value from one
